01 Purpose

任せられるAIロボットを

AIロボットには、人が安心して役割を託せることが求められます。そのためには、変化する実環境を捉え、状況に応じて判断し、リスクを抑えながら安定して動作する力が必要です。 わたしたちは、視覚観測と言語指示から状態とタスクを表現します。人の教示をもとに模倣学習を行い、行動の基盤となるポリシーを構築します。 さらに、そのポリシーを強化学習で最適化するとともに、ワールドモデルで状態遷移を予測します。環境変化に適応する制御方策としてロボットに実装することで、人が安心して役割を託せるAIロボットを実現します。

Learning matters when it works safely in the real world. Our reason to exist

02 Robotics

現実を捉え
制御を極める

01 / Vision Foundation

Machine vision世界を捉える

画像観測から物体、姿勢、運動、関係性を推定し、制御に必要な状態表現へ変換します。

02 / Language Foundation

Language grounding意味をつなぐ

言語指示とタスク文脈を状態表現へ接地し、知覚と行動に共通する条件を構築します。

03 / Imitation Core

Imitation learning人間から学ぶ

教示データに含まれる動作と判断から、制御の初期ポリシーを獲得します。

04 / Reinforcement Core

Reinforcement learning試行で磨く

報酬と環境フィードバックからポリシーを最適化し、未学習条件への適応性能を高めます。

05 / World Model Core

World model先を予測する

観測と行動から状態遷移を学び、将来の状態を予測して、行動選択と制御に活用します。

06 / Safety Core

Safety by design安全を組み込む

AIロボットは動けばいいというものではありません。人と共存する環境でリスクを抑えた行動選択を目指します。

Vision + Language → Imitation → Reinforcement → World Model → Safety → Control
